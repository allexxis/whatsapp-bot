# Whatsapp bot

Create discord like commands "!help" in whatsapp to enable helpers for clients or organizations

## Important details

Bun is only used in this program as a bundler but nodejs is the one that runs the program

## Installation

```bash
bun install
```

## Usage

```bash
bun run dev
```
